## sealer check

check the state of cluster 

```
sealer check [flags]
```

### Examples

```
sealer check --pre or sealer check --post
```

### Options

```
  -h, --help   help for check
      --post   Check the status of the cluster after it is created
      --pre    Check dependencies before cluster creation
```
