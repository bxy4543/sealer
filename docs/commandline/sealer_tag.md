## sealer tag

create a new tag that refers to a local ClusterImage

```
sealer tag [flags]
```

### Examples

```
sealer tag kubernetes:v1.19.8 registry.cn-qingdao.aliyuncs.com/sealer-apps/kubernetes:v1.19.8
```

### Options

```
  -h, --help   help for tag
```
