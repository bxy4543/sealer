## sealer load

load image

### Synopsis

Load an image from a tar archive

```
sealer load [flags]
```

### Examples

```
sealer load -i kubernetes.tar.gz
```

### Options

```
  -h, --help           help for load
  -i, --input string   read image from tar archive file
```
