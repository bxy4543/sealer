## sealer rmi

Remove local images by name or ID

```
sealer rmi [flags]
```

### Examples

```
sealer rmi registry.cn-qingdao.aliyuncs.com/sealer-io/kubernetes:v1.19.9
```

### Options

```
  -f, --force   force removal of the image
  -h, --help    help for rmi
```

### Options inherited from parent commands

